# Assets needed

Place the following files in the correct subdirectories.
Without them the app still works (fallback rendering), but will look better with real assets.

## assets/frames/
| File | Card type |
|---|---|
| frame-effect.png  | Effect Monster (orange) |
| frame-normal.png  | Normal Monster (tan) |
| frame-fusion.png  | Fusion Monster (purple) |
| frame-ritual.png  | Ritual Monster (blue) |
| frame-synchro.png | Synchro Monster (white) |
| frame-xyz.png     | XYZ Monster (black) |
| frame-link.png    | Link Monster (dark blue) |
| frame-spell.png   | Spell Card (green) |
| frame-trap.png    | Trap Card (pink) |

Recommended source: https://github.com/IamIronGolem/yugioh-card-maker (assets/ folder)
Also: https://github.com/Daksh777/YuGiOh-Card-Maker

## assets/icons/
| File | Description |
|---|---|
| attr-dark.png   | DARK attribute circle |
| attr-light.png  | LIGHT attribute circle |
| attr-fire.png   | FIRE attribute circle |
| attr-water.png  | WATER attribute circle |
| attr-earth.png  | EARTH attribute circle |
| attr-wind.png   | WIND attribute circle |
| attr-divine.png | DIVINE attribute circle |
| attr-spell.png  | SPELL icon |
| attr-trap.png   | TRAP icon |
| star.png        | Gold level star (for normal/effect/etc.) |
| star-xyz.png    | Dark rank star (for XYZ) |

Attribute icons can be downloaded from the YuGiOh wiki (yugipedia.com).
Expected size: 32×32 px for attribute circles, 24×24 px for stars.
